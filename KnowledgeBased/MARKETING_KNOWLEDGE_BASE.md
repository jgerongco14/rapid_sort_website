# PANTS: Rapid Sort Marketing Knowledge Base

## Product Snapshot

### Product name
PANTS: Rapid Sort

### Core idea
PANTS stands for Place, Animal, Name, Thing. The app turns that familiar word game into a fast mobile sorting experience where players drag words into the correct category before time runs out.

### Platform
- Flutter mobile app
- Android-ready with Google Play signing setup already documented
- Firebase-backed for cloud features

### One-line pitch
PANTS: Rapid Sort is a fast-paced word sorting game that mixes reflexes, vocabulary, progression, and collectible rewards in a colorful arcade-style format.

### Short marketing summary
Players race against the clock to sort words into the right category, unlock new modes, collect words into sticker books, earn rewards, and track progress over time. The app blends casual wordplay with progression systems like levels, coins, unlocks, themes, and collectible books.

## Main Selling Points

- Familiar concept with a fresh gameplay twist
- Fast rounds that are easy to start and replay
- Multiple game modes with different difficulty styles
- Level progression, XP, coins, and unlockable content
- Sticker Book collection system for long-term retention
- Topic-based Sort Mode for themed knowledge challenges
- Leaderboard support for competitive motivation
- Google sign-in and cloud save support

## Key Gameplay Loop

1. The player sees a word.
2. The player quickly sorts it into the correct category.
3. Correct answers increase score, coins, and XP.
4. Progress unlocks more modes, more content, and more collection rewards.
5. Players return to improve scores, complete books, and unlock more rewards.

## Core Features

### Core word sorting gameplay
- Main categories are Place, Animal, Name, and Thing
- Drag-and-drop interaction
- Speed and accuracy both matter

### Multiple game modes
- Classic Mode: core fast sorting experience
- Sort Mode: themed topic-based sorting
- Zen Mode: slower pace with lives instead of timer pressure
- Blitz Mode: 60-second fast scoring mode
- Chaos Mode: category zones shuffle for added difficulty
- Multiplayer is present in the progression structure as future or expandable content

### Progression system
- XP and level-based unlocks
- Coins earned through gameplay
- Unlock-based content pacing across modes
- Timer upgrades and theme progression

### Loadout and power-ups
- Shield
- Revive
- Skip
- Reward cards and consumables are part of the progression/shop loop

### Sticker Book system
- Library-style collection of books
- Books track discovered words and completion progress
- Book detail pages paginate entries
- Rewards are tied to collection completion

### Sort Mode topics
- Topic selection screen
- Firebase-backed topic data with offline/local fallback behavior
- Examples include country, food, and biology-style categories

### Social and support features
- Leaderboard for score competition
- Profile and account settings
- In-app support ticket flow

## FAQ Draft

### What is PANTS: Rapid Sort?
It is a mobile word sorting game inspired by Place, Animal, Name, Thing, redesigned as a fast arcade-style challenge.

No. It is designed for anyone who enjoys vocabulary, quick thinking, and replayable challenge.

Yes. The app includes several modes such as Classic, Zen, Blitz, Chaos, and themed Sort Mode play.

### Can players track progress?
Yes. The app includes XP, levels, coins, collections, and profile-linked progress systems.

Yes. The app includes Google sign-in and Firebase-based sync and leaderboard features.
